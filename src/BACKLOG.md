# Open Stories:

- The user shall be capable of loading a .png/.jpg into the canvas
- The user shall be capable of seeing info, warn and error dialogs (extend jfxUiService with 3 according methods)

- Eraser function
- Color palette
- Color picker
- Pipette
- Geometric objects
- Upload function
- The user shall be capable of filling all adjacent pixels with the same color with another color (Standard bucket)
- The user shall be capable of replacing all pixels of a given color with another color (replace color)

# Closed Stories:

- The user shall be capable of creating a Canvas
- The user shall be capable of drawing on a Canvas
- The user shall be capable of creating a Canvas with adjustable size
- The user shall be capable of saving the canvas as a .png/.jpg (kein swingfx.util)
- The user Shall be capable of picking a color (evtl. colorpicker)
- The user Shall be capable of changing the color of individual pixels (onDrag)
